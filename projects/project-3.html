نبذة عن المشروع

Project Overview

تم تطوير لوحة معلومات تفاعلية باستخدام Power BI لتحليل بيانات المبيعات وعرض المؤشرات الرئيسية بطريقة بصرية تساعد على فهم الأداء بشكل أسرع وأكثر وضوحًا.

An interactive dashboard was developed using Power BI to analyze sales data and present key indicators visually for faster and clearer performance understanding.

الهدف من المشروع

Business Objective

تحويل بيانات المبيعات الخام إلى مؤشرات ورسوم تفاعلية تساعد في متابعة الأداء، مقارنة المنتجات والفئات، وتحليل الفروع والاتجاهات الزمنية.

To transform raw sales data into interactive KPIs and visuals that support performance tracking, product comparison, branch analysis, and time-based trend exploration.

منهجية التحليلAnalysis Approach

01

تحديد المؤشرات الرئيسية

Defining Key Indicators

التركيز على مؤشرات المبيعات حسب العلامة التجارية، الفئة، الفئة الفرعية، الفروع، وتغير الأسعار زمنيًا.

Focused on sales indicators by brand, category, sub-category, branches, and price changes over time.

02

تحليل البيانات بصريًا

Visual Data Analysis

استخدام الرسوم التفاعلية لتحويل البيانات إلى معلومات سهلة القراءة والمقارنة.

Used interactive visuals to turn data into clear and comparable information.

03

استكشاف الأنماط والاتجاهات

Exploring Patterns & Trends

إبراز العلامات التجارية والفئات الأكثر نشاطًا والفروقات بين الفروع والتغيرات الزمنية.

Highlighted active brands, top categories, branch differences, and time-based changes.

04

دعم اتخاذ القرار

Decision Support

توفير رؤية شاملة تساعد على متابعة المؤشرات واستكشاف البيانات عبر المرشحات التفاعلية.

Provided a comprehensive view to monitor KPIs and explore data through interactive filters.

3 مميزات رئيسية3 Main Features

KPIs Overview

Quick overview of key indicators to understand performance.

Interactive Filters

تصفية البيانات حسب السنة والربع والفروع.

Filter data by year, quarter, and branches.

Visual Comparison

مقارنة المنتجات والفئات والفروع بصريًا.

Visual comparison across products, categories, and branches.

أبرز النتائجKey Insights

تحديد العلامات التجارية الأكثر مساهمة في النشاط البيعي.

Identified the brands with the highest contribution to sales activity.

تحليل الفئات والمنتجات الأعلى من حيث حجم المبيعات والكميات.

Analyzed top categories and products by sales volume and quantities.

مقارنة أداء الفروع المختلفة من خلال عرض بصري واضح.

Compared branch performance through clear visual representations.

متابعة التغيرات الزمنية في الأسعار بين السنوات والفترات المختلفة.

Tracked price changes across available years and periods.
